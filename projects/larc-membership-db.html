LARC Membership Database Project

Database design and implementation for the Lancaster Amateur Radio Club (W2SO)

Overview

This project involved designing and implementing a relational database for managing member and event data for the Lancaster Amateur Radio Club. It includes SQL schema, technical documentation, and presentation materials.

Key Components

  • 📄 Technical Report
  • 📊 Presentation
  • 🧬 SQL Script

Technologies Used

MySQL, ER modeling, normalization, data integrity constraints, user access roles.
