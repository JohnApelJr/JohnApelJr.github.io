OCR with Deep Learning

Convolutional Recurrent Neural Network (CRNN) with MJSynth dataset

Overview

This project implements a CRNN model for text recognition using the MJSynth (Synth90k) dataset. The model was benchmarked against Tesseract and Google Vision AI for accuracy and robustness.

Key Features

  • CRNN Architecture: CNN + BiLSTM + CTC Loss
  • Data Augmentation for scene variation
  • Hyperparameter tuning with validation accuracy tracking
  • Evaluation against commercial OCR tools
